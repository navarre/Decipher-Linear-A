# References

This directory holds source documents ingested for the `minoan-linear-a` project.

## Principles

- Preserve original files unchanged.
- Keep provenance visible.
- Separate core scholarly references from speculative or tangential material.
- Do not treat undeciphered-script claims as established merely because they are written confidently.
- Prefer explicit classification over silent endorsement.

## Layout

- `core/` — high-value primary/secondary scholarly sources directly relevant to Linear A corpus, script, language, or philology.
- `comparative/` — potentially useful comparative linguistic or script material not directly about Linear A.
- `fringe/` — speculative, non-peer-reviewed, or methodologically weak materials kept for awareness but not treated as baseline evidence.
- `manifests/` — inventory and provenance notes for ingested files.
